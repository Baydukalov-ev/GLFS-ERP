Регистр "Операции по факторингу (син)"

Назначение
Этот регистр предназначен для регистрации движений документа "Факторинг (син)".

Расшифровка движений по видам расчетов:
• С дебитором
По данному виду расчетов учитываются наши взаиморасчеты с Дебитором по поставкам, переданным на факторинговое обслуживание. По сути это то же, что и «Взаиморасчеты с контрагентами по документам расчетов» (можно посмотреть типовым отчетом «Ведомость по взаиморасчетам с контрагентами»), а также обороты по бухгалтерскому счету 009.
Движение «Приход» — увеличение долга Дебитора перед нами. Формируется документом «Факторинг. Продажа дебитору».
Движение «Расход» — уменьшение долга Дебитора перед нами. Формируется документами «Факторинг. Оплата дебитором фактору» и «Факторинг. Оплата дебитора нам».
Соответственно, положительное сальдо по данному виду расчетов означает долг Дебитора перед нами; отсутствие сальдо — отсутствие долга Дебитора, а отрицательное сальдо — ошибку при отражении операций (например, не отразили Факторинг. Продажа дебитору).
• С фактором
По данному виду расчетов учитываются наши взаиморасчеты с Фактором по поставкам, переданным на факторинговое обслуживание. По сути это то же, что и «Взаиморасчеты с контрагентами по документам расчетов» (можно посмотреть типовым отчетом «Ведомость по взаиморасчетам с контрагентами»), а также обороты по бухгалтерскому счету 76.09.
Движение «Приход» — увеличение долга Фактора перед нами. Формируется документами «Факторинг. Передача документов фактору» и «Факторинг. Оплата нами фактору».
Движение «Расход» — уменьшение долга Фактора перед нами. Формируется документами «Факторинг. Финансирование фактором», «Факторинг. Перевод остатка фактором», «Факторинг. Вознаграждение фактора» и «Факторинг. Оплата дебитора нам».
Соответственно, положительное сальдо по данному виду расчетов означает долг Фактора перед нами; отсутствие сальдо — отсутствие долга Фактора, а отрицательное сальдо — ошибку при отражении операций.
• По сумме уступленного долга
По данному виду расчетов учитываются наши расчеты с Фактором по сумме уступленного долга.
Движение «Приход» — увеличение суммы уступленного долга. Формируется документом «Факторинг. Передача документов фактору».
Движение «Расход» — уменьшение суммы уступленного долга. Формируется документами «Факторинг. Оплата дебитором фактору» и «Факторинг. Оплата нами фактору».
Соответственно, положительное сальдо по данному виду расчетов означает наличие незакрытого уступленного долга у Фактора (незакрытой накладной у фактора); отсутствие сальдо — отсутствие уступленного долга, а отрицательное сальдо — ошибку при отражении операций.
• По сумме комиссии
По данному виду расчетов учитываются наши взаиморасчеты с Фактором по сумме вознаграждения (комиссий и сборов за факторинговое обслуживание).
Движение «Приход» — увеличение долга Фактора перед нами (т.е. мы заплатили комиссию тогда, когда контрагент заплатил фактору, а счет-фактуру за услуги получим от фактора только в конце месяца). Формируется документами «Факторинг. Оплата дебитором фактору» и «Факторинг. Оплата нами фактору».
Движение «Расход» — уменьшение долга Фактора перед нами. Формируется документом «Факторинг. Вознаграждение фактора».
Соответственно, положительное сальдо по данному виду расчетов означает долг Фактора перед нами; отсутствие сальдо — отсутствие долга Фактора перед нами, а отрицательное сальдо — ошибку при отражении операций.
• По сумме финансирования
По данному виду расчетов учитываются наши взаиморасчеты с Фактором исключительно по суммам финансирования. Очень важный вид расчетов, который нигде больше не фигурирует. Необходим для сверки с Фактором по суммам финансирования.
Движение «Приход» — увеличение нашего долга перед Фактором. Формируется документом «Факторинг. Финансирование фактором».
Движение «Расход» — уменьшение нашего долга перед Фактором. Формируется документами «Факторинг. Оплата дебитором фактору» и «Факторинг. Оплата нами фактору».
Соответственно, положительное сальдо по данному виду расчетов означает наш долг перед Фактором; отсутствие сальдо — отсутствие нашего долга перед Фактором, а отрицательное сальдо — ошибку при отражении операций.
